# Ads On Boards USA — Website Rebuild Proposal

Digital proposal for Ads On Boards USA. Built with Next.js 16, React 19,
Tailwind CSS v4, and GSAP. Deploys to Vercel as-is.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Push to GitHub and import into Vercel — no configuration needed.
